# ACX-Chip-Project
SULI Intern Project for Argonne Summer 2026 - Kailey McGady

The goal of this project it to take a microfluidic chip device provided from ACX Instruments and mix droplets of water on it. By showing the basic ability of the water being able to be mixed, I am hoping to learn how to code the software to create channels that can be used continously and switch up the chemical susbtance I am mixing. The hope is to be able to recreate Argonne National Labs color mixing experiment ran in the RPL and get to a point where I can integrate the AI interface to make the chip experiment fully autonmous. This repo is all of the work I put in during my 10 week internship and the code I used to have the project function.

To contact the company go to https://www.acxinst.com/

As of 6/24 I have working paths and am awaiting a new batch of silicone oil to test the master scripts. Many iterations have been made but the project is on track to be finished on time and successfully!
